# Лабораторная работа №6: Управление базой данных магазина «Золотое Яблоко»  
Embedded SQL (ECPG) + PostgreSQL + C++

## Вариант выполнения
  - Поддержка двух режимов доступа через роли БД (администратор / гость)  
  - Создание новых пользователей БД с назначением роли

## Что реализовано

- Создание и удаление базы данных (`CREATE DATABASE`, `DROP DATABASE`)
- Все операции с данными — **только через хранимые процедуры и функции**
- Запрещено выполнение произвольного SQL из клиентского кода (кроме создания/удаления БД)
- Однотабличная БД товаров (таблица `products`)
- Функционал:
  - Очистка таблицы (`TRUNCATE`)
  - Добавление товара
  - Просмотр всех товаров
  - Поиск по текстовому не ключевому полю (`brand`)
  - Обновление записи по ID
  - Удаление по текстовому полю (`brand`)
- Два уровня доступа на уровне СУБД:
  - `shop_admin` — полный доступ
  - `shop_guest` — только просмотр и поиск
- Возможность создания новых пользователей с выбором роли (`sp_create_user`)
- Консольный интерфейс с понятным меню
